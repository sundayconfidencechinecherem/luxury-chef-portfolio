# User Stories

## High-Net-Worth Client

As a wealthy individual,
I want to see a premium and structured website,
So that I trust the chef’s professionalism before contacting.

---

As a government official,
I want a discreet and formal booking process,
So that my event planning remains confidential.

---

As an event planner,
I want to clearly understand pricing tiers and minimum booking,
So that I do not waste time negotiating.

---

## The Chef (Business Owner)

As the chef,
I want to enforce a minimum booking value,
So that I stop underpricing my services.

---

As the chef,
I want bookings to be automated,
So that I reduce manual communication stress.

---

As the chef,
I want to position myself as luxury,
So that I attract higher-quality clients.

---

## System-Level Stories

As the system,
I must filter low-budget inquiries,
So that only qualified leads reach the chef.

As the system,
I must collect event details upfront,
So that the chef can prepare properly before responding.
