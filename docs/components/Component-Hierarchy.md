# Component Hierarchy – Luxury Chef Booking System

## Public Pages

### Header
- Logo
- Navigation links: Home, About, Services, Booking
- Sticky on scroll

### Footer
- Copyright
- Social links
- Contact info

### HeroSection
- Title / subtitle
- CTA button to booking form
- Background image

### AboutSection
- Chef story
- Certifications
- Media mentions

### ServicesSection
- List of offered services
- Icons / images

### PortfolioSection
- Gallery of past events
- Lightbox modal

### BookingForm
- Full name, email, phone
- Event date & time picker
- Guest count, event type
- Notes textarea
- Submit button
- Inline validation

### TestimonialsSection
- Client quote cards

---

## Admin Pages

### AdminLogin
- Email & password fields
- Login button
- Error alerts

### AdminDashboard
- Summary of bookings
- Filter by status
- Quick links to booking detail

### BookingList
- Table of bookings
- Columns: Name, EventDate, Status, Actions
- Sortable and paginated

### BookingDetail
- Detailed booking info
- Approve / Reject buttons
- Trigger payment link button
- Notes field

### Alerts & Modals
- Confirmation modals
- Success / Error notifications
