# Design System & Visuals – Luxury Private Chef Website

## Typography

| Element       | Font Family        | Size        | Weight | Line Height | Usage                   |
|---------------|-----------------|------------|--------|------------|------------------------|
| H1            | Playfair Display | 48px       | 700    | 1.2        | Page hero titles       |
| H2            | Playfair Display | 36px       | 600    | 1.3        | Section headings       |
| H3            | Inter           | 28px       | 600    | 1.3        | Subsection headings    |
| Body / P      | Inter           | 18px       | 400    | 1.5        | Paragraphs / content   |
| Small / Caption | Inter         | 14px       | 400    | 1.4        | Footnotes / labels     |

---

## Color Palette

| Role            | Hex       | Usage                                      |
|-----------------|-----------|-------------------------------------------|
| Primary         | #0A0A0A   | Dark luxury background / header           |
| Secondary       | #C49B63   | Accent buttons, links, highlights         |
| Neutral Light   | #F8F8F8   | Background panels / forms                  |
| Text Primary    | #1A1A1A   | Main body text                             |
| Text Secondary  | #555555   | Subtext / captions                         |
| Success / Paid  | #4CAF50   | Booking success / confirmation             |
| Error / Rejected| #E53935   | Booking rejection alerts                   |

---

## Spacing & Layout

- **Container width:** 1200px max for desktop  
- **Grid:** 12-column layout  
- **Margins / Padding:** 24px base, scale by 8px increments  
- **Section spacing:** 80px vertical spacing between main sections  

---

## Buttons

| Type          | Background  | Text Color | Border Radius | Usage                       |
|---------------|------------|-----------|---------------|-----------------------------|
| Primary       | #C49B63    | #FFFFFF   | 8px           | Call-to-action buttons      |
| Secondary     | #0A0A0A    | #FFFFFF   | 8px           | Admin actions / secondary   |
| Disabled      | #E0E0E0    | #AAAAAA   | 8px           | Disabled state             |
| Hover States  | Slight opacity or color shift for all buttons |

---

## Forms & Inputs

- Input border: 1px solid #CCCCCC, border-radius: 6px  
- Focus state: border-color #C49B63, box-shadow subtle  
- Labels: Inter, 14px, bold  

---

## Imagery

- Use professional photography where available  
- Stock images (Pixabay / Unsplash) only as placeholder  
- Hero images full-width, dark overlay for text readability  

---

## Components Interaction Guidelines

- Buttons: hover and focus highlight  
- Modals: centered, dimmed backdrop  
- Forms: inline validation + error messages  
- Admin dashboard: sortable tables, paginated booking lists
